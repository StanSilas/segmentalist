Segmentalist
============

Author
------
Herman Kamper, kamperh@gmail.com.


Overview
--------
Perform unsupervised acoustic word segmentation using both unigram and bigram
language models.


Dependencies
------------
- [Cython](http://cython.org/)


Building the code
-----------------
Run `make`.
